# 🗂️ Inventory Management System

Manage your business inventory efficiently with this web application.

## ✨ Features

- 🔐 User Authentication
- 📦 Product Management
- 📊 Inventory Tracking
- 🔒 User-Specific Data
- 🛡️ Admin Panel

## 🛠️ Technologies

- **Backend:** Spring Boot, Thymeleaf, MongoDB
- **Frontend:** Tailwind CSS, Thymeleaf Templates
